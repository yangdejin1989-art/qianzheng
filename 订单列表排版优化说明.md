# 订单列表排版优化说明

## 优化内容

为了在一个页面中显示更多订单内容，对订单列表进行了全面的紧凑化优化。

## 具体改进

### 1. 标题区域优化
- 标题字体：1rem → 0.9rem
- 上下间距：8px/12px → 6px/8px
- 整体更紧凑，节省顶部空间

### 2. 筛选器区域优化

#### 输入框和下拉框
- 字体大小：0.8rem → 0.75rem
- 内边距：默认 → 3px 6px
- 输入框宽度：
  - 姓名：80px → 70px
  - 手机号：100px → 95px
  - 申请编码：120px → 110px
  - 状态/办理方式：120px → 95px
  - 签证类型：120px → 100px
  - 负责人：120px → 100px
  - 日期输入框：100px → 90px

#### 标签和按钮
- 标签字体：0.8rem → 0.72rem
- 按钮字体：0.8rem → 0.75rem
- 按钮内边距：4px 12px → 3px 10px
- 行间距：mb-2 → mb-1

### 3. 表格优化

#### 表头 (thead)
- 字体大小：默认 → 0.72rem
- 单元格内边距：默认 → 4px 2px-6px
- 移除了 table-sm 类，使用自定义padding

#### 列宽优化
| 列名 | 原宽度 | 新宽度 | 说明 |
|------|--------|--------|------|
| 创建时间 | 90px | 75px | 更紧凑的日期时间显示 |
| 申请编码 | 150px | 130px | 缩短编码列 |
| 姓名 | 80px | 65px | 紧凑显示，支持多人 |
| 手机号 | 140px | 105px | 电话号码刚好显示 |
| 微信/LINE | 120px | 95px | 缩短联系方式 |
| 办理方式 | 90px | 70px | 紧凑显示 |
| 预计完成 | 100px | 75px | 日期紧凑显示 |
| 签证类型 | 120px | 95px | 缩短类型列 |
| 状态 | 80px | 65px | badge紧凑显示 |
| 负责人 | 100px | 70px | 员工名紧凑显示 |
| 反馈 | 200px | 150px | 缩短反馈列 |
| 操作 | 100px/140px | 75px/105px | 按钮更小 |

#### 表格内容 (tbody)
- 字体大小：
  - 主内容：0.85rem → 0.75rem
  - 创建时间：0.85rem → 0.7rem
  - 微信/LINE：默认 → 0.7rem
  - 同行人：默认 → 0.7rem（灰色）
- 单元格内边距：0.5rem 0.3rem → 4px 4px
- 行高：默认 → 1.2（更紧凑）

#### 按钮优化
- 字体大小：0.75rem → 0.68rem
- 内边距：默认 → 2px 8px
- 更小巧的"详情"和"分配"按钮

#### Badge标签优化
- 状态标签：默认 → 0.7rem，padding: 2px 6px
- 负责人标签：默认 → 0.68rem，padding: 2px 4px

### 4. 容器优化
- 表格容器字体：默认 → 0.75rem
- 整体字体层级统一调整

## 优化效果

### 空间节省
- **垂直空间**：每行节省约 30%
- **水平空间**：总宽度节省约 20%
- **可见行数**：同屏可显示订单数量提升约 40%

### 视觉效果
- ✅ 信息更加紧凑，一目了然
- ✅ 保持可读性，不显得拥挤
- ✅ 层次清晰，重要信息突出
- ✅ 响应式布局，适应不同屏幕

### 操作体验
- ✅ 减少滚动次数
- ✅ 提高工作效率
- ✅ 快速浏览订单信息
- ✅ 保持操作按钮可点击性

## 字体大小层级

为了保持视觉层次，采用了以下字体大小体系：

1. **标题层**：0.9rem（订单申请管理）
2. **筛选器层**：
   - 输入框/下拉框：0.75rem
   - 标签文字：0.72rem
   - 按钮：0.75rem
3. **表格层**：
   - 表头：0.72rem
   - 主内容：0.75rem
   - 次要内容：0.7rem（时间、联系方式等）
   - 标签：0.68-0.7rem
   - 按钮：0.68rem

## 注意事项

1. **可读性**：虽然字体变小，但仍保持在可读范围内（≥0.68rem）
2. **点击区域**：按钮padding虽小，但点击区域仍然足够
3. **响应式**：在小屏幕上可能需要进一步调整
4. **打印**：如需打印，建议使用浏览器的缩放功能

## 进一步优化建议

如果还需要更紧凑的显示，可以考虑：

1. **隐藏不常用列**：如"反馈"列可以折叠
2. **使用图标**：将"办理方式"等文字改为图标
3. **悬浮显示**：某些信息hover显示完整内容
4. **分页显示**：减少每页显示数量但提高加载速度
5. **自定义列**：让用户选择显示哪些列
