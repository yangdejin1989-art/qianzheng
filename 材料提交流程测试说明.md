## 🚀 运行测试

### 1. 安装依赖
```bash
# 自动安装依赖
node install_deps.js

# 或手动安装
npm install axios puppeteer
```

### 2. 启动服务器
```bash
# 启动后端服务器
cd server
npm start

# 启动前端服务器（新开命令行窗口）
cd client
npm start
```

### 3. 运行测试

#### 运行所有测试
```bash

```

```bash
# 只运行后端材料提交流程测试
node test_material_flow.js
```

#### 单独运行前端测试
```bash
# 只运行前端按钮功能测试
node test_frontend_buttons.js
```

## 🐛 常见问题解决

### 1. 后端测试失败
- **问题**: 找不到套餐数据
- **解决**: 确保数据库中有套餐数据，或先运行 `debug_test_simple.js` 创建测试数据

### 2. 前端测试失败
- **问题**: 找不到按钮元素
- **解决**: 确保前端服务器正常运行，检查页面是否正确加载

### 3. 材料提交失败
- **问题**: 400/500 错误
- **解决**: 检查后端服务器状态和数据库连接

### 4. 按钮点击无响应
- **问题**: 按钮不可点击
- **解决**: 检查按钮状态和页面加载情况
