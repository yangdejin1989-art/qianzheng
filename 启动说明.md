## 📧 邮箱配置

### Gmail配置（推荐）
1. 登录Gmail：https://gmail.com
2. 进入Google账户管理：右上角头像 → "管理您的Google账户"
3. 左侧选择"安全性"
4. 开启"两步验证"
5. 找到"应用专用密码"
6. 生成邮件应用密码
7. 复制16位密码（格式：abcd efgh ijkl mnop）

### QQ邮箱配置
1. 登录QQ邮箱：https://mail.qq.com
2. 设置 → 账户 → 开启SMTP服务
3. 发送短信获取授权码
4. 复制16位授权码

## ❗ 常见问题

### 1. 端口被占用
- 脚本会自动检测并清理端口
- 手动清理：`taskkill /f /im node.exe`

### 2. MongoDB连接失败
- 确保MongoDB服务已启动
- 检查连接字符串：mongodb://localhost:27017/kdai_broadband

### 3. 邮件发送失败
- 检查邮箱配置是否正确
- Gmail：确认两步验证已开启
- QQ邮箱：确认SMTP服务已开启

### 4. 前端无法访问
- 等待npm start完全启动（约30秒）
- 检查3000端口是否被占用

## 🛠️ 开发模式

如需开发调试，可以手动启动：

```bash
# 启动后端
cd server
set EMAIL_USER=your-email@gmail.com
set EMAIL_PASS=your-app-password
node index.js

# 启动前端（新终端）
cd client
npm start
```
