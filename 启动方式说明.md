# 季舒签证申请系统 - 启动方式说明

## 🚀 启动方式概览

本系统提供三种邮箱验证启动方式，您可以根据需要选择：

### 1. Gmail邮箱启动 (一键启动.bat)
- **适用场景**: 使用Gmail邮箱发送验证码
- **优势**: 国际化服务，功能完善
- **缺点**: 可能遇到网络连接问题
- **文件**: `一键启动.bat`

### 2. QQ邮箱启动 (QQ邮箱一键启动.bat)
- **适用场景**: 使用QQ邮箱发送验证码
- **优势**: 国内网络稳定，连接速度快
- **缺点**: 需要配置授权码
- **文件**: `QQ邮箱一键启动.bat`

### 3. 163邮箱快速启动 (jishu163邮箱快速启动.bat) ⭐
- **适用场景**: 使用已验证的163邮箱快速启动
- **优势**: 无需输入授权码，一键启动，配置已验证
- **缺点**: 仅适用于特定邮箱账户
- **文件**: `jishu163邮箱快速启动.bat`

## 📧 邮箱配置说明

### Gmail配置
1. 开启两步验证
2. 生成应用专用密码
3. 使用应用专用密码（不是登录密码）

### QQ邮箱配置
1. 开启SMTP服务
2. 获取授权码（16位字符）
3. 使用授权码（不是QQ密码）

### 163邮箱配置
1. 开启SMTP服务
2. 获取授权码（16位字符）
3. 使用授权码（不是登录密码）

## 🔧 使用方法

### 首次使用
1. 运行对应的配置助手脚本
2. 按照提示配置邮箱信息
3. 运行对应的启动脚本

### 日常使用
- 直接运行对应的启动脚本即可
- 系统会自动检测邮箱类型
- 自动配置SMTP参数
- **推荐使用**: `jishu163邮箱快速启动.bat`（已验证配置）

## 📁 文件说明

| 文件名 | 用途 | 说明 |
|--------|------|------|
| `一键启动.bat` | Gmail启动脚本 | 默认使用Gmail邮箱 |
| `QQ邮箱一键启动.bat` | QQ邮箱启动脚本 | 专门用于QQ邮箱 |
| `jishu163邮箱快速启动.bat` | 163邮箱快速启动 | 已验证配置，一键启动 |
| `Gmail配置助手.bat` | Gmail配置帮助 | 指导Gmail配置 |
| `QQ邮箱配置助手.bat` | QQ邮箱配置帮助 | 指导QQ邮箱配置 |
| `启动方式说明.md` | 使用说明文档 | 本文档 |

## ⚠️ 注意事项

1. **不要同时运行多个启动脚本**
2. **保持服务窗口运行，关闭即停止服务**
3. **首次使用需要配置邮箱信息**
4. **163邮箱推荐国内企业用户使用**
5. **QQ邮箱推荐国内个人用户使用**
6. **Gmail推荐海外用户使用**

## 🧪 测试验证

启动完成后，可以通过以下方式测试：

1. 访问前端页面: http://localhost:3000
2. 点击"申请进度查询"
3. 输入手机号和邮箱
4. 点击"发送验证码"
5. 检查邮箱是否收到验证码

## 🔍 故障排除

### 常见问题
- **连接超时**: 检查网络连接，尝试使用163邮箱或QQ邮箱
- **认证失败**: 检查邮箱配置，确认密码/授权码正确
- **端口占用**: 脚本会自动清理端口，如仍有问题请重启电脑
- **163邮箱问题**: 163邮箱使用25端口，某些网络可能被封锁，可尝试使用465端口

### 获取帮助
如遇到问题，请查看服务窗口的日志信息，或联系技术支持。
