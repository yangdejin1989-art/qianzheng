# ✏️ 签证类型编辑功能说明

## 新增功能
现在可以直接编辑已添加的签证类型，无需删除后重新添加！

## 功能特点

### 1. **编辑按钮** 🔧
每个已添加的签证类型旁边都有"编辑"和"删除"两个按钮。

### 2. **编辑模式** ✏️
点击"编辑"按钮后：
- 该签证类型会高亮显示（黄色背景）
- 其数据会自动填充到下方的输入框
- "添加"按钮变为绿色的"更新"按钮
- 出现一个"✕"取消按钮
- 其他签证类型的"编辑"按钮会被禁用

### 3. **更新操作** 💾
修改完成后：
- 点击"更新"按钮保存修改
- 或点击"✕"按钮取消编辑
- 或点击其他签证类型的"编辑"按钮（会自动放弃当前编辑）

### 4. **视觉反馈** 👁️
- **编辑中的项**：黄色背景 + 金色边框
- **正常项**：白色背景
- **更新按钮**：绿色（编辑模式）
- **添加按钮**：蓝色（正常模式）
- **提示信息**："✏️ 正在编辑签证类型，修改后点击'更新'按钮"

## 使用步骤

### 编辑现有签证类型

1. **找到要修改的签证类型**
   ```
   例如：单次 ¥12000/次 ¥15000 (日元)
   ```

2. **点击"编辑"按钮**
   - 该项会变成黄色背景
   - 下方输入框自动填充该项的数据

3. **修改数据**
   - 类型名称：如改为"单次入境"
   - 币种：如改为"¥ 人民币"
   - 价格：如改为 15000
   - 原价：如改为 18000

4. **保存修改**
   - 点击绿色的"更新"按钮
   - 该项立即更新

5. **或取消编辑**
   - 点击"✕"按钮
   - 或点击其他项的"编辑"按钮

### 添加新签证类型

如果没有在编辑任何项，按正常流程添加：
1. 填写类型、币种、价格、原价
2. 点击蓝色的"添加"按钮

## 技术实现

### 新增状态
- `editingVisaTypeIndex`：记录正在编辑的签证类型索引

### 新增函数
- `editVisaType(index)`：开始编辑指定索引的签证类型
- `cancelEditVisaType()`：取消当前编辑
- 修改 `addVisaType()`：根据是否在编辑模式，执行添加或更新操作

### UI变化
- 编辑中的项添加黄色背景和边框
- 其他项的编辑按钮在编辑模式下被禁用
- "添加"按钮在编辑模式下变为"更新"按钮
- 显示/隐藏取消按钮
- 显示编辑提示信息

## 完整操作流程示例

### 示例1：修改币种
```
原始数据：
单次  ¥日元  12000  15000

操作步骤：
1. 点击"编辑" → 数据填充到输入框
2. 修改币种：¥日元 → ¥人民币
3. 点击"更新" → 保存成功

结果：
单次  ¥人民币  12000  15000
```

### 示例2：修改价格和原价
```
原始数据：
一年多次  ¥人民币  15000  18000

操作步骤：
1. 点击"编辑"
2. 修改价格：15000 → 16000
3. 修改原价：18000 → 20000
4. 点击"更新"

结果：
一年多次  ¥人民币  16000  20000
```

### 示例3：取消编辑
```
1. 点击某项的"编辑"按钮
2. 修改了一些数据
3. 改变主意了，点击"✕"按钮
4. 输入框清空，黄色高亮消失
5. 原数据保持不变
```

## 注意事项

1. **一次只能编辑一项**
   - 编辑某项时，其他项的"编辑"按钮会被禁用
   - 必须先完成或取消当前编辑

2. **删除优先级**
   - 即使在编辑模式，仍然可以删除任何项
   - 删除正在编辑的项会自动取消编辑

3. **数据验证**
   - 类型和价格是必填项
   - 原价可选
   - 币种默认为"人民币"

4. **自动状态管理**
   - 保存套餐成功后自动退出编辑模式
   - 取消套餐编辑时自动退出编辑模式
   - 新增套餐时自动清空编辑状态

## 优势

✅ **更方便**：无需删除后重新添加  
✅ **更直观**：黄色高亮清楚显示正在编辑哪一项  
✅ **更安全**：编辑模式下其他项被保护，不会误操作  
✅ **更灵活**：可以随时取消编辑
