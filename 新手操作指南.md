# 🚀 邮箱验证码功能 - 新手操作指南

## 📋 完整操作流程（第一次使用）

### 步骤一：配置邮箱服务 ⚙️

#### 选项A：使用Gmail（推荐）

1. **登录Gmail**
   - 打开 https://gmail.com
   - 登录您的Gmail账号

2. **开启两步验证**
   - 点击右上角头像 → "管理您的Google账户"
   - 左侧点击"安全性"
   - 找到"两步验证"，点击开启
   - 按提示完成手机验证

3. **生成应用专用密码**
   - 在"安全性"页面，找到"应用专用密码"
   - 选择应用：邮件
   - 选择设备：Windows计算机
   - 点击"生成"
   - **重要**：复制生成的16位密码（如：abcd efgh ijkl mnop）

4. **设置环境变量**
   - 按 `Win + R`，输入 `cmd`，按回车
   - 在命令提示符中输入：
   ```cmd
   set EMAIL_USER=您的Gmail地址
   set EMAIL_PASS=刚才复制的16位密码
   ```
   
   **示例**：
   ```cmd
   set EMAIL_USER=test@gmail.com
   set EMAIL_PASS=abcdefghijklmnop
   ```

#### 选项B：使用QQ邮箱

1. **登录QQ邮箱**
   - 打开 https://mail.qq.com
   - 登录您的QQ邮箱

2. **开启SMTP服务**
   - 点击"设置" → "账户"
   - 找到"POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"
   - 开启"SMTP服务"
   - 按提示发送短信，获取授权码

3. **设置环境变量**
   ```cmd
   set EMAIL_USER=您的QQ邮箱@qq.com
   set EMAIL_PASS=刚才获取的授权码
   ```

### 步骤二：启动系统 🚀

#### 方法A：使用自动启动脚本（推荐）

1. **双击运行**
   - 找到项目文件夹中的 `start-with-email.bat`
   - 双击运行

2. **检查配置**
   - 脚本会自动检查邮箱配置
   - 如果配置正确，会自动启动前后端服务

#### 方法B：手动启动

1. **启动后端服务**
   ```cmd
   cd kdai-broadband\server
   node index.js
   ```

2. **启动前端服务**（新开一个命令提示符窗口）
   ```cmd
   cd kdai-broadband\client
   npm start
   ```

### 步骤三：测试功能 🧪

1. **打开浏览器**
   - 访问 http://localhost:3000

2. **进入查询页面**
   - 点击"申请进度查询"按钮

3. **测试验证流程**
   - **第一步**：输入查询信息
     - 手机号：输入任意手机号（如：13800138000）
     - 申请编码：可以留空
     - 邮箱地址：输入您的真实邮箱地址
     - 点击"发送验证码"

   - **第二步**：邮箱验证
     - 查收邮件（可能在垃圾邮件文件夹）
     - 输入6位验证码
     - 点击"验证并查询"

   - **第三步**：查看结果
     - 如果有匹配的申请记录，会显示详细信息
     - 如果没有，会提示"未找到申请记录"

## 🔍 常见问题解决

### 问题1：邮件发送失败

**现象**：点击"发送验证码"后提示发送失败

**解决方案**：
1. 检查邮箱配置是否正确
2. 确认网络连接正常
3. 检查Gmail是否开启了两步验证
4. 确认应用专用密码是否正确

### 问题2：收不到验证码邮件

**解决方案**：
1. 检查垃圾邮件文件夹
2. 等待1-2分钟（邮件可能有延迟）
3. 确认邮箱地址输入正确
4. 尝试重新发送验证码

### 问题3：验证码验证失败

**解决方案**：
1. 确认验证码输入正确（6位数字）
2. 检查验证码是否过期（10分钟有效期）
3. 尝试重新获取验证码

### 问题4：服务启动失败

**解决方案**：
1. 确认Node.js已安装
2. 检查端口5000和3000是否被占用
3. 运行 `npm install` 安装依赖
