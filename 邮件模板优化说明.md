# 邮件模板优化说明

## 📝 优化时间
2025年10月30日

## 🎯 优化目标
将原来宽带业务的邮件模板优化为签证办理业务专用模板

## ✨ 主要改进

### 1. 业务字段更新
将宽带业务相关字段改为签证业务字段：

| 原字段 | 新字段 |
|--------|--------|
| 网络类型 | 办理方式 |
| 安装日期 | 预计完成 |
| 套餐 | 签证类型 |
| 基本信息 + 服务信息 | 申请信息（合并） |

### 2. 管理员反馈改为可选
**重要改进：** 管理员反馈不再自动显示在所有邮件中

- **不显示反馈的状态：** 待处理
- **可选显示反馈的状态：** 待确认、处理中、已完成、已取消

**使用方式：**
- 只有在管理员填写了"管理员反馈"字段时，邮件中才会显示"💬 重要提示"区块
- 如果反馈为空，则该区块不会出现

### 3. 各状态邮件优化

#### 📋 待处理
- **标题：** 签证申请已收到
- **内容：** 告知申请已提交，客服将在1-2工作日联系
- **优化点：** 
  - 明确后续流程步骤
  - 提醒准备证件材料
  - 不显示管理员反馈

#### 📝 待确认
- **标题：** 请补充签证材料
- **内容：** 提示客户补充材料，加快办理进度
- **优化点：**
  - 去掉硬编码的材料清单
  - 引导客户通过链接查看或咨询客服
  - 可显示管理员的补充说明

#### ⚡ 处理中
- **标题：** 材料审核通过，正在办理中
- **内容：** 告知材料通过，正在办理签证手续
- **优化点：**
  - **移除了"安装团队"、"安装日期"等宽带字样**
  - 改为"办理签证申请手续"
  - 列出办理进度步骤
  - 提醒保持联系畅通

#### 🎉 已完成
- **标题：** 恭喜！您的签证办理已完成
- **内容：** 祝贺办理完成，感谢客户信任
- **优化点：**
  - 更友好的完成祝贺
  - 提供后续支持说明

#### ❌ 已取消
- **标题：** 您的签证申请已取消
- **内容：** 告知取消，提供重新申请支持
- **优化点：**
  - 更友好的表述
  - 提供客服支持渠道

### 4. 视觉优化
- 申请编码加粗高亮显示
- 签证类型使用彩色标签
- 当前状态使用圆角徽章
- 优化了间距和排版

## 📊 邮件发送规则

### 发送模式
- **完全手动发送**：管理员需点击"发送状态邮件"按钮
- **不会自动发送**：状态变更不会触发自动发送

### 反馈显示逻辑
```javascript
// 待处理 - 不显示反馈
${renderFeedbackSection(context)}  // showFeedback默认false

// 待确认、处理中、已完成、已取消 - 可选显示反馈
${renderFeedbackSection(context, true)}  // 有内容才显示
```

### 何时显示"💬 重要提示"
只有同时满足以下条件时才显示：
1. 状态为：待确认/处理中/已完成/已取消
2. 管理员填写了"管理员反馈"字段
3. 反馈内容不为空

## 💡 使用建议

### 待处理状态
- 客户刚提交申请时发送
- 不需要填写管理员反馈
- 让客户知道申请已收到

### 待确认状态
- 需要客户补充材料时发送
- **建议在"管理员反馈"中说明：**
  - 具体缺少哪些材料
  - 特殊注意事项
  - 补充材料的截止时间

### 处理中状态
- 材料审核通过后发送
- **可在"管理员反馈"中说明：**
  - 预计完成时间
  - 需要客户注意的事项
  - 或留空不填

### 已完成状态
- 签证办理完成后发送
- **可在"管理员反馈"中说明：**
  - 领取方式
  - 注意事项
  - 或留空不填

### 已取消状态
- 申请被取消时发送
- **建议在"管理员反馈"中说明：**
  - 取消原因
  - 退款说明（如适用）
  - 重新申请指引

## 🔧 技术说明

### 修改的文件
- `qianzheng/server/emailTemplates/statusTemplates.js`

### 主要函数
1. `renderInfoBlocks(context, statusLabel, timeLabel, timeValue)` - 渲染申请信息区块
2. `renderFeedbackSection(context, showFeedback = false)` - 渲染管理员反馈（可选）
3. `buildStatusEmail(status, context)` - 构建完整邮件内容

### 邮件内容来源
```javascript

  applyCode: '申请编码',
  name: '客户姓名',

  address: '地址',
  email: '邮箱',

  networkType: '办理方式',      // 原"网络类型"
  installDate: '预计完成日期',  // 原"安装日期"

  feedback: '管理员反馈',       // 可选显示
  submitTime: '提交时间',

  linkUrl: '快捷链接（待确认状态）'

```
