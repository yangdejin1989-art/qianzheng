# 签证系统使用说明

## 一、系统简介

本系统是一个签证申请管理平台，包含前台客户网站和后台管理功能。客户可以浏览签证套餐、在线提交申请材料；管理员可以管理网站内容、审核签证申请。

## 二、启动与停止

### 日常启动
- **一键启动.bat** — 同时启动后端（端口 5000）和前端（端口 3000）
- **停止所有服务.bat** — 停止所有运行中的服务

### 访问地址
- 前台网站：http://localhost:3000
- 后台管理：http://localhost:3000/admin

## 三、系统架构

```
qianzheng/
├── client/              # 前端 - React 应用
├── server/              # 后端 - Express 服务器
│   ├── config/          # 配置文件
│   ├── middleware/      # 中间件（认证、权限）
│   ├── models/          # 数据模型（14个）
│   ├── routes/          # API 路由
│   ├── utils/           # 工具函数
│   ├── uploads/         # 图片存储
│   └── index.js         # 服务器入口
├── package.json         # 项目配置
├── vercel.json          # Vercel 部署配置
└── .gitignore           # Git 忽略配置
```

### 技术栈
- **前端**：React + Axios
- **后端**：Node.js + Express
- **数据库**：MongoDB Atlas（云端）
- **图片存储**：本地文件系统（server/uploads/）
- **邮件**：Nodemailer（163 邮箱 SMTP）

## 四、核心功能

### 客户端
1. 浏览签证套餐和产品
2. 在线填写签证申请
3. 上传申请材料
4. 查看申请状态

### 管理后台
1. **套餐管理** — 添加/编辑/删除签证套餐
2. **申请管理** — 查看和审核签证申请
3. **轮播图管理** — 管理首页轮播图片
4. **公告管理** — 发布和管理公告
5. **FAQ 管理** — 编辑常见问题
6. **页脚管理** — 编辑网站页脚信息
7. **网站设置** — 全局网站配置
8. **用户管理** — 管理用户账户和权限
9. **统计面板** — 查看业务统计数据

## 五、数据库

### 连接配置
配置文件：`server/.env`

```
MONGO_URI=mongodb://用户名:密码@主机:27017/visa-system?ssl=true&authSource=admin&directConnection=true&serverSelectionTimeoutMS=15000
```

### 关键参数
| 参数 | 说明 |
|------|------|
| ssl=true | 使用 SSL/TLS 加密 |
| directConnection=true | 直连单节点，跳过副本集发现 |
| serverSelectionTimeoutMS=15000 | 服务器选择超时 15 秒 |

### 数据模型（14个）
| 模型 | 用途 |
|------|------|
| Admin | 管理员账户 |
| Application | 签证申请单 |
| Package | 签证套餐 |
| Carousel | 轮播图 |
| Notice | 公告 |
| FAQ | 常见问题 |
| Footer | 页脚信息 |
| SiteSettings | 网站设置 |
| MaterialTemplate | 材料模板 |
| QuestionTemplate | 问题模板 |
| Statistics | 统计数据 |
| Blacklist | IP 黑名单 |

## 七、图片上传

- 上传接口：`POST /api/upload`
- 存储位置：`server/uploads/`
- 访问路径：`/uploads/文件名`
- 支持格式：PNG、JPG、GIF 等常见图片格式

## 八、常见问题

### 1. 启动后无法访问
- 检查端口 5000 和 3000 是否被占用
- 确认后端和前端都已成功启动

### 2. 数据库连接失败
- 检查 `server/.env` 中的 MONGO_URI 配置
- 确认网络连接正常（特别是使用代理时）
- `directConnection=true` 参数可避免代理兼容问题

### 3. 图片无法显示
- 检查图片是否成功上传到 `server/uploads/` 目录
- 确认图片路径格式正确（以 `/uploads/` 开头）
