# 订单状态扭转测试使用说明

## 🚀 快速开始

### 1. 安装依赖
```bash
# 方法1: 使用自动安装脚本
node install_deps.js

# 方法2: 手动安装
npm install axios puppeteer
```

### 2. 启动服务器
```bash
# 启动后端服务器
cd server
npm start

# 启动前端服务器（新开一个命令行窗口）
cd client
npm start
```

### 3. 运行测试
```bash
# 简化调试测试（推荐先运行）
node debug_test_simple.js

# 调试测试
node debug_test.js

# 前端调试
node debug_frontend.js

# 完整测试
node run_tests.js
```

## 🐛 常见问题解决

### 1. 依赖安装失败
```bash
# 如果 puppeteer 安装慢，可以跳过 Chromium 下载
set PUPPETEER_SKIP_CHROMIUM_DOWNLOAD=true
npm install puppeteer
```

### 2. 前端页面加载失败
- 检查前端服务器是否运行在 `http://localhost:3000`
- 检查后端服务器是否运行在 `http://localhost:5000`
- 查看生成的截图文件 `debug-frontend.png` 和 `page-error.png`

### 3. 后端接口测试失败
- 检查数据库连接
- 检查是否有套餐数据
- 查看后端控制台错误信息

### 4. 表单提交测试失败
- 检查后端接口是否正常
- 检查网络连接状态

1. **404 错误** - 检查接口路径和数据
2. **500 错误** - 检查后端日志和数据库
3. **表单提交失败** - 检查后端接口和网络连接
4. **页面加载超时** - 检查服务器状态和网络
