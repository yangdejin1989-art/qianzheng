# 韩国签证（日本出发）配置说明

**⚠️ 重要：本文件记录了韩国签证的配置信息，请勿删除！**

## 📋 配置概览

- **签证类型**: 韩国签证
- **适用范围**: 东京管辖区（东京、千叶、埼玉、茨城、栃木、群马在住）
- **申请条件**: 在日本居住满2年并持有中长期签证
- **申请时间**: 每周二、周四申请
- **办理时长**: 顺利情况下工作日5-6天
- **多次签证**: 申请者必须曾持有过单次签证
- **材料数量**: 8个材料项
- **问题数量**: 8个问题

---

## 📑 材料清单（8项）

### 必须材料（4项）

#### 1. passport - 护照原件
- **ID**: `passport`
- **说明**: 有效期6个月以上的护照所有页面拍照
- **是否必须**: ✅ 是
- **允许多张**: ✅ 是
- **注意事项**: 需拍摄所有页面，包括空白页

#### 2. photo_4.5x3.5 - 证件照片
- **ID**: `photo_4.5x3.5`
- **说明**: 4.5cm×3.5cm，白色背景，3个月内拍摄
- **是否必须**: ✅ 是
- **允许多张**: ❌ 否
- **注意事项**: 
  - 尺寸：4.5cm（高）× 3.5cm（宽）
  - 背景必须是白色
  - 必须是3个月内拍摄的

#### 3. residence_card - 在留卡
- **ID**: `residence_card`
- **说明**: 在留卡正反面清晰照片，有效期3个月以上
- **是否必须**: ✅ 是
- **允许多张**: ✅ 是
- **注意事项**: 
  - 必须拍摄正反两面
  - 照片必须清晰可见
  - 有效期必须在3个月以上

#### 4. residence_certificate - 住民票
- **ID**: `residence_certificate`
- **说明**: 全家族成员，全部事项不能省略（除个人番号外）
- **是否必须**: ✅ 是
- **允许多张**: ❌ 否
- **注意事项**: 
  - 必须包含全家族成员
  - 全部事项都必须显示
  - 只有个人番号可以省略

### 追加材料（4项 - 根据身份情况提供）

#### 5. bank_statement - 银行通帐
- **ID**: `bank_statement`
- **说明**: 最近6个月的银行通帐复印件（非永住者必须）
- **是否必须**: ⚠️ 非永住者必须
- **允许多张**: ✅ 是
- **适用对象**: 非永住者

#### 6. employment_certificate - 在职证明书
- **ID**: `employment_certificate`
- **说明**: 技术·人文知识·国际业务资格在留期间1年持有者需提交
- **是否必须**: ⚠️ 特定人群必须
- **允许多张**: ❌ 否
- **适用对象**: 技术·人文知识·国际业务资格在留期间1年持有者

#### 7. student_certificate - 在学证明或学生证
- **ID**: `student_certificate`
- **说明**: 4年制大学留学生提供在学证明书或学生证正反面复印件
- **是否必须**: ⚠️ 留学生必须
- **允许多张**: ✅ 是
- **适用对象**: 4年制大学留学生

#### 8. family_relation - 家族关系证明
- **ID**: `family_relation`
- **说明**: 留学生无本人名义财政证明时，提供本国父母财政资料需附此证明
- **是否必须**: ⚠️ 特定情况必须
- **允许多张**: ✅ 是
- **适用对象**: 留学生（无本人名义财政证明，使用父母财政资料时）

## ❓ 问题清单（8个问题）

### 1. 中国的地址
- **ID**: `china_address`
- **类型**: 文本输入
- **是否必须**: ✅ 是
- **说明**: 请填写申请人在中国的详细地址

### 2. 日本的手机电话号码
- **ID**: `japan_phone`
- **类型**: 文本输入
- **是否必须**: ✅ 是
- **说明**: 请填写在日本的联系电话号码

### 3. 配偶者信息
- **ID**: `spouse_info`
- **类型**: 多行文本
- **是否必须**: ✅ 是
- **填写格式**:
  ```
  如有配偶，请填写：
  姓名：
  生日：
  电话号码：
  
  如无配偶请填写"无"
  ```

### 4. 最终学历信息
- **ID**: `education_info`
- **类型**: 多行文本
- **是否必须**: ✅ 是
- **填写格式**:
  ```
  学校名称：
  学校地址：
  ```

### 5. 现在工作情况
- **ID**: `employment_info`
- **类型**: 多行文本
- **是否必须**: ✅ 是
- **填写格式**:
  ```
  现在是否有工作：
  公司名称：
  公司地址：
  公司电话：
  
  如无工作请填写"无"
  ```

### 6. 过去5年内韩国旅行记录
- **ID**: `korea_travel_history`
- **类型**: 多行文本
- **是否必须**: ✅ 是
- **填写格式**:
  ```
  是否去过韩国：
  目的：
  出发日期：
  回程日期：
  
  如没有去过请填写"无"
  ```
- **注意**: 多次签证申请者必须曾持有过单次签证

### 7. 过去5年内其他国家旅行记录
- **ID**: `other_travel_history`
- **类型**: 多行文本
- **是否必须**: ✅ 是
- **填写格式**:
  ```
  去过的国家名（除中国、韩国外）：
  目的：
  出发日期：
  回程日期：
  
  如没有去过请填写"无"
  ```

### 8. 预计出发时间
- **ID**: `departure_date`
- **类型**: 文本输入
- **是否必须**: ✅ 是
- **格式**: YYYY-MM-DD
- **示例**: 2024-05-20

## 💡 Material ID 设计说明

### 材料ID命名规则
- `passport`: 护照（通用ID，与其他签证类型一致）
- `photo_4.5x3.5`: 韩国签证特定尺寸的照片
- `residence_card`: 在留卡（日本特有材料）
- `residence_certificate`: 住民票（日本特有材料）
- `bank_statement`: 银行流水/通帐（通用ID）
- `employment_certificate`: 在职证明（通用ID）
- `student_certificate`: 学生证明（通用ID）
- `family_relation`: 家族关系证明（通用ID）

### ID一致性策略
使用通用ID（如`passport`、`bank_statement`）的材料，在切换签证类型时可以智能保留数据。

## ⚠️ 重要注意事项

### 申请条件
1. **居住要求**: 在日本居住满2年
2. **签证类型**: 必须持有中长期签证
3. **管辖区域**: 仅适用于东京管辖区

### 有效期要求
1. **护照**: 有效期6个月以上
2. **在留卡**: 有效期3个月以上

### 申请时间
- **受理日期**: 每周二、周四
- **办理时长**: 工作日5-6天（顺利情况下）

### 多次签证
- 申请多次签证的前提：必须曾经持有过单次韩国签证

### 非永住者追加要求
如果不是永住者，必须提供：
1. 最近6个月的银行通帐
2. 在职证明书（特定签证类型）
3. 留学生需提供在学证明或学生证
