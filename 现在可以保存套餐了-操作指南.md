## 📝 操作步骤

### 1. 刷新页面
- 按键盘 **F5** 刷新浏览器页面
- 或者点击浏览器地址栏的刷新按钮

### 2. 进入套餐管理
- 登录后台管理系统
- 进入"套餐管理"页面

### 3. 添加/编辑套餐

#### 基本信息
- **套餐名称**：如"台湾签证"
- **预计完成时间**：如"2-5个工作日"
- **排序**：数字越小越靠前

#### 签证类型与价格 ⭐ 重点
这是新的统一价格管理区域：

1. **类型**：输入签证类型名称
   - 例如：单次、一年多次、三年多次

2. **币种**：选择该类型使用的货币
   - ¥ 人民币（CNY）
   - ¥ 日元（JPY）
   - $ 美元（USD）
   - € 欧元（EUR）

3. **价格**：输入价格数字
   - 例如：12000

4. **原价**：（可选）输入划线价
   - 例如：15000

5. 点击 **"添加"** 按钮

6. **可以添加多个类型**，每个类型可以用不同的币种！

**示例**：
```
类型：单次        币种：¥日元    价格：12000   原价：15000
类型：一年多次    币种：¥人民币  价格：15000   原价：18000
```

#### 其他信息
- **简短描述**：一句话介绍
- **特色功能**：
  - 输入功能描述
  - 点击"添加"
  - 可以添加多个
  - 这些会显示在套餐卡片底部
- **详细说明**：使用富文本编辑器编辑详细内容
- **套餐图片**：建议尺寸 800x600px

### 4. 保存
- 点击 **"保存修改"** 或 **"新增套餐"**
- 应该会立即成功！
- 套餐列表会自动刷新
