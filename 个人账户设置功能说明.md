# 个人账户设置功能说明

## 📋 功能概述

新增了个人账户设置功能，允许所有用户（管理员和员工）管理自己的账户信息和密码。

---

## ✨ 功能特点

### 1. 账户信息显示
- 显示当前用户名
- 查看账户状态

### 2. 基本信息修改
- ✅ 修改显示名称
- ✅ 修改手机号码
- ✅ 修改邮箱地址
- ✅ 修改微信号
- ✅ 修改QQ号
- ✅ 实时保存更新
- ✅ 自动同步到本地存储

### 3. 密码修改
- ✅ 需要输入当前密码验证
- ✅ 新密码长度验证（至少6位）
- ✅ 二次确认新密码
- ✅ 加密存储（bcrypt）

### 4. 权限控制优化
- ✅ 日志管理：仅管理员可访问
- ✅ 数据统计：仅管理员可访问
- ✅ 员工账户不显示这些功能菜单

---

## 🎯 使用方法

### 访问个人设置

1. 登录后台管理系统
2. 在页面**右上角**找到用户名按钮
3. 点击用户名，会弹出下拉菜单
4. 选择 **"个人设置"** 进入

> 💡 **注意**：所有用户（管理员和员工）都可以访问此功能，无需特殊权限。

### 下拉菜单选项
- **个人设置**：管理账户信息、联系方式、修改密码
- **退出登录**：安全退出系统

---

## 🔧 功能详解

### 修改显示名称

1. 在"基本信息"卡片中
2. 修改"显示名称"输入框
3. 点击"保存基本信息"按钮
4. 系统会自动更新并显示成功提示

**显示名称用途**：
- 在右上角用户菜单中显示
- 管理员登录：还会在侧边栏顶部显示
- 员工登录：管理员可在"用户管理"中查看（用于备注）

---

### 修改密码

1. 在"修改密码"卡片中填写：
   - **当前密码**：您现在使用的密码
   - **新密码**：新的密码（至少6位）
   - **确认新密码**：再次输入新密码

2. 点击"修改密码"按钮

3. 系统验证：
   - ✅ 当前密码是否正确
   - ✅ 新密码长度是否符合要求
   - ✅ 两次输入的新密码是否一致

4. 修改成功后，表单自动清空

**安全提示**：
- 定期修改密码，保护账户安全
- 密码建议包含字母和数字
- 不要与他人共享密码
- 如发现账户异常，立即修改密码

## 🔌 技术实现

### 前端组件
- **文件**：`qianzheng/client/src/AccountSettings.js`
- **功能**：
  - 账户信息展示
  - 基本信息修改表单（显示名称、联系方式）
  - 密码修改表单
  - 消息提示和加载状态

### 后端API

#### 1. 更新基本信息
```
PUT /api/admin/profile
Authorization: Bearer {token}
Body: {
  "displayName": "新的显示名称",
  "phone": "手机号",
  "email": "邮箱",
  "wechat": "微信号",
  "qq": "QQ号"
}
```

#### 2. 修改密码
```
PUT /api/admin/change-password
Authorization: Bearer {token}
Body: {
  "currentPassword": "当前密码",
  "newPassword": "新密码"
}
```

### 数据库
- 使用现有的 `Admin` 模型
- 字段：`displayName`, `phone`, `email`, `wechat`, `qq`, `password`, `updatedAt`
- 密码使用 bcrypt 加密存储
- 联系方式字段均为可选

## 🚀 权限控制

### 访问权限
- ✅ **所有用户**都可以访问个人设置
- ✅ 不需要任何特殊权限
- ✅ 每个用户只能修改自己的信息

### 安全验证
- 通过 JWT token 识别当前用户
- 修改密码需要验证当前密码
- 只能修改自己的账户，无法越权

---

## 📊 使用场景

### 管理员
1. 修改自己的显示名称（会在侧边栏显示）
2. 定期修改密码
3. 更新个人信息

### 员工
1. 修改显示名称（管理员可见，用于备注）
2. 修改自己的登录密码
3. 更新联系方式（手机、邮箱、微信、QQ）

---

## ⚠️ 注意事项

1. **密码长度**：新密码至少6位，建议8位以上
2. **当前密码**：修改密码时必须正确输入当前密码
3. **密码确认**：两次输入的新密码必须一致
4. **立即生效**：密码修改后立即生效，下次登录需要使用新密码
5. **保持登录**：修改密码不会退出当前登录状态

---

## 🐛 常见问题

### Q1: 忘记密码怎么办？
**A**: 联系管理员使用"重置管理员密码.bat"脚本重置密码

### Q2: 修改密码后需要重新登录吗？
**A**: 不需要，当前登录状态会保持，下次登录时使用新密码

### Q3: 显示名称和用户名有什么区别？
**A**: 
- **用户名**：登录使用，不可修改
- **显示名称**：显示用的昵称，可以修改
